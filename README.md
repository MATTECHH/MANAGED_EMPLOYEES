# 👨‍💼 Managed Employees

Sistema fullstack para **gerenciamento de funcionários**, desenvolvido com foco em **Clean Code**, **componentização**, **custom hooks** e **boas práticas com APIs REST**.

## 🚀 Demo

👉 https://mattechh.github.io/MANAGED_EMPLOYEES/

---

## ✨ Funcionalidades

### Backend

✅ CRUD completo
✅ Validação com Regex
✅ Criptografia com bcrypt
✅ Persistência com Prisma ORM
✅ Tratamento de erros HTTP

### Frontend

✅ Cadastro e listagem de funcionários
✅ Custom Hooks
✅ Navegação SPA
✅ Animações com Framer Motion
✅ Interface responsiva

---

## 🛠 Stack

**Frontend:** React • React Router • Tailwind • Framer Motion
**Backend:** Node.js • Express • Prisma • Bcrypt
**Database:** PostgreSQL • MySQL • SQLite

---

## 📡 API

```http
GET    /users
GET    /users/:id
POST   /users
PUT    /users/:id
DELETE /users/:id
```

---

## ⚙️ Executando localmente

```bash
git clone https://github.com/MATTECHH/MANAGED_EMPLOYEES.git

cd MANAGED_EMPLOYEES

npm install

npx prisma generate
npx prisma migrate dev

node server.js
npm run dev
```
