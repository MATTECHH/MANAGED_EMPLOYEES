# 👨‍💼 Managed Employees

Sistema completo para **gerenciamento de funcionários**, desenvolvido com foco em organização, produtividade e boas práticas de desenvolvimento.

O projeto permite cadastrar, visualizar, editar e remover colaboradores de forma prática e intuitiva, simulando um ambiente corporativo real para administração de equipes.

---

## 🚀 Demonstração do Projeto

🔗 Repositório oficial:  
https://github.com/MATTECHH/MANAGED_EMPLOYEES

---

## 📌 Sobre o projeto

O **Managed Employees** foi desenvolvido com o objetivo de praticar conceitos fundamentais e avançados de desenvolvimento fullstack, incluindo:

- CRUD completo
- Organização em camadas
- Componentização
- Gerenciamento de estado
- Consumo/manipulação de dados
- Boas práticas de arquitetura
- Responsividade

Este projeto simula um sistema real de RH/ERP para gestão de colaboradores.

## ✨ Funcionalidades

✅ Cadastro de funcionários

## ⚙️ Como executar localmente

### 1 Clone o repositório

```bash
git clone https://github.com/MATTECHH/MANAGED_EMPLOYEES.git
```

### 2 Entre na pasta

```bash
cd MANAGED_EMPLOYEES
```

### 3 Instale as dependências

```bash
npm install
```

### 4 Execute o projeto

```bash
npm run dev
```
